# Javascript-practice

## Settings
- `npm init -y`
- `npm install parcel-bundler-D`
- package.json
  - dev : parcel index.html로 개발 서버를 열겠다.
  - build : 실제로 만든 결과물을 dist 폴더에 만들겠다.
  ```
  "scripts": {
      "dev": "parcel index.html", 
      "build": "parcel build index.html"
    },
  ```
- `npm run dev` : Server Open!
